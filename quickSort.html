Quick Sort Visualization

This program makes takes a input of numbers sepearated by commas and shows how the quickSort algorithm sorts numbers. The display shows the partitioning of the algorithm by indexing an array and selecting a pivot element. Index i increments through array until reaching a number greater than the pivot element, then index j decrements through the array before reaching a number smaller than the pivot. This repeats while i <= j.

Color Combinations Key:

    i incrementor
    j incrementor
    pivot element
    swap elements

Enter 1-10 numbers sepearated by commas without spaces below
