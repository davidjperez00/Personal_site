Computer Vision Handwriten Math Equation Interpreter

For this project a multi label neural network was designed to interpret and solve a handwritten equation. A preprocessing pipeline was also created to extract digits/operators from the input image and determine which operator/digit is in the extracted image. These predictions are then strung together to produce the solved equation.

Quick Sort Visual

This project is a visualization of the Quick Sort Algorithm. The implementation that I went with paritions from the middle of the inputed array. The visual shows where the incrementers start for each parition and color code when the data in the array gets swapped.

Quick Sort Visual Page

Dodger Game

For this project I decided to create a mini dodger game. The objective of the game is to avoid falling asteroids by moving left and right. At the start of the game a player score and level is shown and increments until a player collides with an asteroid resulting in a a "Game Over!" banner being displayed.
