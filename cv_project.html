Computer Vision Handwriten Math Equation Interpreter

From a high level view this project takes in an image of a basic arithmetic equation and outputs the solution to the equation in the image. For instance, suppose an image contains the handwritten equation “5 + 3”. The first step in identifying and solving the equation is identifying characteristics in the image that are either digits or arithmetic operators using a cv2 contour. From here, the image is run through a preprocessing pipeline to convert it into a black and white 28x28 pixel image resembling the MNIST digits dataset as this is the bulk of the training data. After the image has been processed and extracted, a neural network is used to make a prediction on what digit or operator is contained within each image. The images are then sorted based on their position in the original image from left to right. Lastly, the resulting equation, preprocessed images, and inputted image with bounding boxes where the digits and operators were extracted are displayed as output.
